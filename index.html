Code Retreat

  • Учиться у напарника
  • Расширять зону комфорта
  • Экспериментировать
  • Пробовать новые инженерные практики

  • Не пытаться успеть любой ценой

Условия

  • Одина задача на все сессии
  • Пять сессий по 45 минут
  • В парах
  • Каждую сессию пары меняются
  • В конце сессии весь код удаляется

Полезные советы

  • Пишите тесты
  • Учитесь новому через ограничения!
  • Прочувствовать важнее, чем доделать

Игра «Жизнь»

  • Бесконечное двумерное клеточное поле
  • Клетка бывает живой (черная) или мертвой (белая)
  • Пошаговая игра
    • Менее двух живых соседей → смерть
    • Более трех живых соседей → смерть
    • Ровно три живых соседа → жизнь

< 2 соседей → смерть
> 3 соседей → смерть
= 3 соседа → жизнь
= 2 соседа → состояние не меняется
